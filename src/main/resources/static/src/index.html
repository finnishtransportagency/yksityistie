Tervetuloa Digiroadin tilastotyökaluun.

Työkalun avulla voit tarkastella tietolajien päivityksiä haluamasi ajanjakson

aikana yhdessä tai useammassa kunnassa. Tulokset esitetään interaktiivisena

graafina ja tiedot voi ladata palvelusta taulukoksi .CSV -muodossa.

  1. Aseta aikaväli, jolta aineiston muutokset haetaan
  2. Valitse tietolaji sekä 1-10 tarkasteltavaa kuntaa
  3. Voit suodattaa tietoja hallinnollisella luokalla
  4. Muuta kaavion esitystapoja ON-OFF painikkeilla. Lisätietoikkuna aukeaa, kun viet hiiren kaavion päälle ja voit zoomata kaaviota hiiren rullalla
  5. Lataa tiedot taulukkona koneellesi

Ei tuloksia? Kaikkia tietolajeja ei ylläpidetä jatkuvasti eli kokeile hakea pidemmältä aikaväliltä.

Tapahtui virhe? Paina uudelleen Hae tiedot–painiketta!
